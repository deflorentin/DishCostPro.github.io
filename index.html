How to use DishCost Pro
A quick guide to getting accurate food cost results.

Step 1 — Ingredient Price List

Upload a CSV or Excel file with your ingredient costs. Required columns:

  • Ingredient Name — e.g. "Chicken Breast"
  • Unit — e.g. "kg", "liter", "loaf"
  • Unit Cost — the cost per unit (number only, no currency symbols)
  • Quantity on Hand — optional — your current stock count. Shows an inventory overview panel with low-stock alerts (≤3 units highlighted in red).

Step 2 — Recipe Data

One row per ingredient per dish. Required columns:

  • Dish Name — e.g. "Grilled Chicken"
  • Ingredient Name — must match exactly to your ingredient list
  • Quantity Used — amount used per serving (number)
  • Unit — same unit as in your ingredient list

Step 3 — Processing

Set your target food cost percentage (industry standard: 28–35%), choose your currency, and select a markup method. Then click Calculate.

Export Options

After calculation, export results as a CSV spreadsheet, Excel file, or a branded PDF report with color-coded rows.

Tips

  • Ingredient names must match exactly between both files (case-insensitive)
  • Use the sample templates as a starting point
  • Units don't affect cost — they're for reference only
